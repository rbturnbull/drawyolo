.. start-quickstart

Draws boxes on images from annotations in YOLO format

Installation
==================================

Install using pip:

.. code-block:: bash

    pip install git+https://github.com/rbturnbull/drawyolo.git


Usage
==================================

See the options with the command:

.. code-block:: bash

    drawyolo --help


.. end-quickstart
